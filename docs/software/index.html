Software

R packages developed and maintained by the R Computing Lab, mostly built around genetically-informed research designs — pedigrees, kinship, and family-based models — without requiring twin samples.

Maintainer

  • ggpedigree — Visualizing pedigrees with ggplot2 and plotly.
  • BGmisc — Extended behavior genetics analysis: relatedness, pedigree construction, and variance component tools.
  • discord — Functions for discordant kinship (sibling/cousin comparison) modeling.
  • NlsyLinks — Utilities and kinship links for research with the National Longitudinal Survey of Youth (NLSY).

Contributor

  • ACEsimFit — ACE kin-pair data simulation and model fitting.

Bug reports and contributions are welcome on each package’s GitHub repo, linked from its CRAN page above.
