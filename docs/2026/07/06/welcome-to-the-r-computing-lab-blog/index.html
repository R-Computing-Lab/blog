Welcome to the R Computing Lab Blog

2026-07-06

This blog is where the R Computing Lab writes about the R side of our work: package releases and updates, tutorials on the genetically-informed methods we build tools for, and shorter notes on reproducible research practice.

A few things to expect here:

  • Package news — updates to BGmisc, ggpedigree, discord, and NlsyLinks — new features, breaking changes, and the reasoning behind them.
  • Tutorials — worked examples using these packages: building pedigrees, running discordant kinship comparisons, and similar family-based designs, in R.
  • Research notes — shorter posts connecting new papers or preprints back to the code and data behind them.

See the Software page for the full package list and the Publications page for the research these tools support. Posts are also available via RSS and on GitHub.
